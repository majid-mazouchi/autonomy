# Autonomy

Personal engineering blog by Majid Mazouchi — notes on control and machine learning. Covers control systems, motor control, machine learning, neural networks, and reinforcement learning. Jekyll-based, hosted on GitHub Pages with zero build configuration.

Features: light/dark mode (auto), inline LaTeX math (MathJax), syntax-highlighted code blocks, reading-progress bar, related-posts suggestions, client-side search, SEO/OpenGraph tags, RSS, sitemap, custom 404.

Live at **https://majid-mazouchi.github.io/autonomy/**
